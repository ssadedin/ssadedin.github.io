Package: gngs

[Groovy] Class OrderedPairActor

              • gngs.OrderedPairActor

              • An actor that adds asynchronous writing to an OrderedPairWriter

                Note that if you feed reads directly via the << operator, you may feed them faster than the underlying writer can write, which will cause the writes to be queued. This queue can expand, taking a large amount of memory. To limit memory, use the {@link#addAlignmentPair} method which will block if the number of pending writes execeeds {@link#maxPendingWrites}.
