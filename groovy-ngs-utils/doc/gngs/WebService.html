[Groovy] Class WebService

    • gngs.WebService

    • Base class for APIs accessing Web services based on JSON data

      Usage: 

       WebService w = new WebService("http://localhost:8000","some/url/path")
 println "Result of call is: " + w.get(foo:10,bar:"tree")
