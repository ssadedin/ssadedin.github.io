Package: gngs

[Groovy] Class FASTQ

    • gngs.FASTQ

    • Simple support for reading fastq files.

      This class supports reading either single end reads (via eachRead(Closure)) or paired end reads (via eachPair(String, String, Closure).

      Gzipped files are supported transparently - if the file name ends in 'gz' then the file is assumed to be gzipped and decoded as such.

      This class only supports reading FASTQ reads, not writing. However the individual reads can be written via the FASTQRead class.
