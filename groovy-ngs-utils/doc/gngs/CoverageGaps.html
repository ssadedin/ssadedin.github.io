Package: gngs

[Groovy] Class CoverageGaps

    • gngs.CoverageGaps

    • Discovers blocks of low coverage in a file output in BEDTools Coverage format.

      Each low coverage block is output as a CoverageBlock object which has statistics about the coverage within the block.

      To use this class, create the object first by passing a file to compute coverage for, then call the calculate method to actually calculate the coverage blocks: 

       CoverageGaps gaps = new CoverageGaps("test.cov.gz")
 gaps.calculate()
