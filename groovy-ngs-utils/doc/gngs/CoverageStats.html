Package: gngs

[Java] Class CoverageStats

      • gngs.CoverageStats 

      • public class CoverageStats
extends graxxiaIntegerStats

        An efficient method to calculate percentiles of coverage values that doesn't require holding them all in memory or sorting them.

        It relies on some limitations regarding coverage values:

      • They are integers
      • We usually know the approximate range, and don't care about it above a certain value. For example, we are unlikely to observe values above 1000 and those that do are unlikely to affect the 50'th percentile.

        simon.sadedin@mcri.edu.au
