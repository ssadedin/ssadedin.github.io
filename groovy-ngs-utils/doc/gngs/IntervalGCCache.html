Package: gngs

[Groovy] Class IntervalGCCache

    • gngs.IntervalGCCache

    • Efficiently supports computing and caching approximate GC content for random fragments of DNA in a genome.

      Efficient caching is performed by quantizing the computation to bins of small size (default 10 bases). This returns a reasonable approximation of the fraction of GC content while reducing storage for cached value by a factor of the binSize squared (100 by default) and increasing the likelihood of cache hits by the same factor.

      The cache must be used by first calling the cache method to register an interval of interest for which fragments are to be queried. This will create an internal backing cache of the size of the interval squared (with additional padding to allow for fragments crossing the interval boundary). The cache method returns a queryable object which can return approximate GC content for any DNA fragment within the interval.

      Acccuracy of the GC content fraction returned is to within approximately 1%. This is primarily due to the use of bytes to store the actual values rather than floats or doubles.
