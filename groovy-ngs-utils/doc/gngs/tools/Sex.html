[Groovy] Class Sex

      • gngs.tools.Sex 

      • @groovy.util.logging.Log
class Sex
extends ToolBase

        Estimates the sex of a sample from one or more VCF file and prints to the console.

        Usage: 

        gngs Sex <vcf file>
        See gngs.VCF#guessSex(int) for details of how sex is determined.

        Simon Sadedin
