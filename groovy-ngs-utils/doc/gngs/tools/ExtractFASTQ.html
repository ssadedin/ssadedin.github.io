Package: gngs.tools

[Groovy] Class ExtractFASTQ

    • gngs.tools.ExtractFASTQ

    • Extract reads as pairs from a sorted BAM and writes them out to standard output in interleaved format.

      Example:

      gngstool ExtractFASTQ -bam my.bam -L chr1:5000000-6000000 > interleaved.fastq An example of realigning a portion of a BAM file: gngstool ExtractFASTQ -bam my.bam -L chr1:5000000-6000000 | \ bwa mem -p ref.fa - | \ samtools view -bSu - | \ samtools sort -o out.bam

      Simon Sadedin
