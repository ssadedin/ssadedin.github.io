        • static void table(java.util.Map options = [:], java.util.List<java.lang.String> headers, java.util.List<java.util.List> rows)

          A utility to print a table of values in a nice format for output on a terminal. Columns are aligned, padded, borders drawn etc. The output format is compatible with markdown for downstream re-formatting into documents.

          By default, the table is printed to stdout. To print it somewhere else, set the out option to a Writer object.

          Parameters:
          headers - a list of column names
          rows - a list of lists, where each inner list represents a row in the table
