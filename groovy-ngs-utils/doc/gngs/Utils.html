          Creates a Reader from any object that can be reasonably interpreted to represent a file, allowing for the possibility that it might be gzipped or bgzipped.

          If a groovy.lang.Closure is provided, then the closure will be called, passing the reader as the first parameter. The return value of this function will be the result of executing the closure. If, however, a Closure is not provided, the return value will be the created java.io.Reader object to be used.

          Returns:

        • static void table(java.util.Map options = [:], java.util.List<java.lang.String> headers, java.util.List<java.util.List> rows)

          A utility to print a table of values in a nice format for output on a terminal. Columns are aligned, padded, borders drawn etc. The output format is compatible with markdown for downstream re-formatting into documents.

          By default, the table is printed to stdout. To print it somewhere else, set the out option to a Writer object.

          Optional parameters:

        • indent: amount to indent table by
        • format: Map keyed on column containing custom formatters
        • topborder: if true, a top border will be added
        • out: Custom output writer / stream to write results to
          Parameters:
          headers - a list of column names
          rows - a list of lists, where each inner list represents a row in the table

        • static java.lang.Object withWriters(java.util.List<java.lang.String> fileNames, groovy.lang.Closure c)

          Executes the given closure, passing as parameters all of the given filenames opened as Writers using intelligent interpretation of the filenames - eg: if ending in .gz, use gzip, etc.
